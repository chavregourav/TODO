# React To-Do List

A simple To-Do List application built with React and Vite. This project allows users to add and delete tasks using React's `useState` hook.

## Features

- Add new tasks
- Delete existing tasks
- Simple and responsive UI
- Built with React Hooks (`useState`)

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- CSS

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-app.git
```

2. Navigate to the project folder:

```bash
cd todo-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```
http://localhost:5173
```

## Usage

1. Type a task into the input field.
2. Click the **Add** button to add it to the list.
3. Click the **Delete** button to remove a task.
